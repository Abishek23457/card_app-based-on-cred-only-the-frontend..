# CRED Inspired React Website

A responsive CRED-inspired fintech landing page built with React, Vite, and custom CSS.

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://127.0.0.1:5173
```

## One-click open on Windows

Double-click:

```text
open-project.bat
```

It installs dependencies if needed, builds the project, opens the browser, and starts the preview server.

## Production build

```bash
npm run build
npm run preview
```

Preview:

```text
http://127.0.0.1:4173
```

## Sections included

- Fullscreen premium hero section
- Members-only trust/about section
- Feature cards for bill payments, rewards, score tracking, UPI, coins, and cashback
- NeoPOP-inspired glossy card and button styling
- Ratings/member statistics
- Security section
- Footer links
